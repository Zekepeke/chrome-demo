Quick Demo Extension

Fill these to personalize the demo and (optionally) hook up a tiny backend.

Used when sending messages to the backend.
Point this to any WebSocket server that echoes JSON. We send {type:'hello'}, {type:'status'}, and your pings. If you leave it blank, the extension still works without a server.

Want a quick backend? Run this in a terminal:


# Node:
# npm i ws
# node -e "new (require('ws').Server)({port:3000}).on('connection',s=>s.on('message',m=>s.clients.forEach(c=>c.send(m))))"
